# Buckshot-Roulette-V2.2.0-STEAM-for-Windows-x86-32-on-OpenGL
I created this repository with good intentions only. This is simply a fan-made build for Windows x86-32 on OpenGL. I make no claim to the developer's rights, fame, or credit. The official developer is Mike Klubnika. For this build to work, you need the game on Steam and OpenGL 3.3 or above support.
Download the exe file and place it in the game's Steam folder, for example: "C:\SteamLibrary\steamapps\common\Buckshot Roulette\Buckshot Roulette_windows\Buckshot Roulette.exe". Replace the original "Buckshot Roulette.exe" file and don't change the name. Now you can open the game from Steam or just the file, and it will be x86-32!
