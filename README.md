# Converting-Godot-based-games-to-other-OS-Architectures-Graphics-engines
I created this repository with the best of intentions. These are simply fan-made versions of the games; I make no claims to the developer's rights, fame, or authorship.

For Buckshot Roulette V2.2.0 (STEAM) for Windows x86-32-on-OpenGL:
  Download the exe file and replace original in the game's Steam folder and don't change the name, for example, in the end it will be: "C:\SteamLibrary\steamapps\common\Buckshot Roulette\Buckshot Roulette_windows\Buckshot Roulette.exe". Now you can open the game from Steam    or just the file, and it will be x86-32!
  You can Download the game file from Releases.
