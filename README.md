# Buckshot-Roulette-V2.2.0-STEAM-for-Windows-x86-32
I created this repository with good intentions only. This is simply a fan-made build for Windows x86-32. I make no claim to the developer's rights, fame, or credit. The official developer is Mike Klubnika. For this build to work, you need the game on Steam.
Download the zip archive and place it in the game's Steam folder, for example: "C:\SteamLibrary\steamapps\common\Buckshot Roulette". Unzip the zip and replace the original "Buckshot Roulette_windows" folder (don't delete and paste it, just paste it over the main one) with the folder from the archive and don't change the name. Now you can open the game from Steam or just the file, and it will be x86-32!
